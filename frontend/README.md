# Sync frontend (static)

1. **Put your landing page here as `index.html`**  
   Copy your file `sync-marketplace (5).html` into this folder and rename it to **`index.html`**.

2. **Deploy on Vercel**  
   Follow the steps in **`../VERCEL-DEPLOY.md`** (in the project root).

This folder already contains `vercel.json` so that all routes serve `index.html`.
